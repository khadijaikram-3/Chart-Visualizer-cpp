# Chart Visualizer (C++)

A graphics-based chart visualizer developed in C++ using graphics.h.
This project supports 10 different chart types for data visualization.

## Charts Implemented
- Bar Chart
- Pie Chart
- Line Chart
- Donut Chart
- Scatter Plot
- Area Chart
- Column Chart
- Histogram
- Stacked Bar Chart
- Multi-Line Chart

## Technologies Used
- C++
- graphics.h (BGI)
- Turbo C++ / WinBGIm

## How to Run
1. Install a compiler that supports graphics.h (Turbo C++ or WinBGIm)
2. Open the source file from `src/main.cpp`
3. Compile and run the program
4. Enter input values as prompted
